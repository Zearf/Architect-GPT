# **App Name**: Architect GPT

## Core Features:

- Codebase Input Module: Allows users to upload or input the JSON payload containing file paths and content of their software project for analysis.
- Semantic Analysis Engine: A generative AI tool that performs structural inference (identifying entry points, core engines), logic decomposition (uncovering 'Jobs-to-be-Done'), and conducts an 'Anti-Slop Audit' by identifying areas for modularity improvements or technical debt, explaining architectural philosophy and data flow across module boundaries.
- SYSTEM_ARCHITECTURE.md Generator: A generative AI tool that synthesizes a comprehensive 'SYSTEM_ARCHITECTURE.md' document based on the semantic analysis, structuring the output with 'Executive Intent', 'Modular Blueprint', 'Reasoning Trace' for complex logic flows, and 'Anti-Slop Audit' findings.
- Architectural Document Viewer: Displays the generated 'SYSTEM_ARCHITECTURE.md' document in a clear, formatted, and easily readable manner within the web application.
- Document Export Functionality: Enables users to download the generated 'SYSTEM_ARCHITECTURE.md' document for offline review and sharing.

## Style Guidelines:

- A dark color scheme is used to evoke a sense of deep analysis and provide visual comfort for extensive text reading. The primary accent color is a bright, intellectual blue (#8CD3FF), symbolizing clarity and insight. The background color is a heavily desaturated, very dark blue-grey (#17181A), offering a professional canvas. A vibrant cyan (#19DBDB) serves as an accent for interactive elements and highlights, ensuring good contrast.
- Headlines utilize 'Space Grotesk' (sans-serif), lending a modern, tech-oriented, and structured feel suitable for architectural headings. Body text and longer explanations employ 'Inter' (sans-serif) for its high legibility and clean, objective appearance. Code snippets and technical listings are presented in 'Source Code Pro' (monospace) to maintain precise formatting.
- Clean, crisp, and functional vector icons that support navigation and communicate complex concepts related to architecture, data flow, and code structure.
- A minimalist, multi-column layout with ample spacing, emphasizing content readability and organization, especially for the detailed architectural documents. Key information will be presented clearly, often with designated sections for each part of the analysis.
- Subtle, deliberate animations on UI interactions, such as input confirmation or document generation progress, to provide visual feedback without distraction, reinforcing a sense of professionalism and efficiency.
